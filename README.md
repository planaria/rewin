# rewin
A Windows GUI library for C++

## Features
- Header-only  
  `#include <rewin/rewin.hpp>`

## Requirements

### Supported Compilers
- Visual Studio 2015 (>= Update 3)

### External Libraries
- Windows Template Library
- Boost C++ Libraries (>= 1.63.0)
- [Catch](https://github.com/philsquared/Catch) (for test)
